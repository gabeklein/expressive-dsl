
<h1 align="center">Expressive-React</h1>

<h4 align="center">
    A collection of Babel plugins that introduce a new syntax<br/>for writing clean, meaningful, <i>expressive</i> React markup.
</h4>

# Install

```
npm install babel-preset-expressive-react
```

**.babelrc**

```
{
    "presets": [
        "expressive-react"
    ]
}
```
